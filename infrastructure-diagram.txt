PORT CONFIGURATION SUMMARY:
════════════════════════════════
┌─ Port 80  (HTTP)  ──► nginx ──► HTTPS Redirect
┌─ Port 443 (HTTPS) ──► nginx ──► Application (Port 3000)
┌─ Port 22  (SSH)   ──► EC2 Management Access
┌─ Port 3306 (MySQL) ──► RDS via SSM Tunnel (Local Port 10003)

DATA FLOW EXAMPLE:
═══════════════════
1. User visits: https://legal.vtc.systems/nda-analyzer
2. nginx (Port 443) ──► Routes to EC2 Application (Port 3000)
3. Application ──► Redirects to Azure AD for authentication
4. Azure AD ──► Validates user & returns to callback URL
5. Application ──► User uploads PDF to S3 bucket
6. Application ──► Stores metadata in MySQL RDS
7. Application ──► Processes document & returns results
